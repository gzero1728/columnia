# Columnia

React 애플리케이션을 위한 완전히 커스터마이즈 가능한 헤드리스 테이블 컴포넌트입니다.
기본 스타일 없이 렌더링 함수를 통해 모든 UI 요소를 직접 제어할 수 있으며, 컬럼 선택 및 재정렬 기능을 포함합니다.

## 주요 기능

- 📊 컬럼 선택 테이블

  - 컬럼 선택/해제 기능
  - 컬럼 순서 변경 (드래그 앤 드롭)
  - 선택된 컬럼 설정 저장/불러오기 (localStorage)

- 🎨 스타일링 제어

  - 원하는 스타일링 솔루션 사용 가능
  - 모든 UI 요소에 대한 완전한 제어권

## 설치

```bash
npm install columnia
# 또는
yarn add columnia
```
